Asterix conversion and validation tool

Pre-build 64-bit linux binary is available to download from this link.

Simply download and copy the file to some location and set file mode, for example: 


sudo cp converter /usr/local/bin
sudo chmod 755 /usr/local/bin/converter
converter --help
